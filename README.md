# 🎓 Online Course Enrollment Manager

A desktop-based **Student Course Management System** built using Python and Tkinter.  
This project demonstrates the use of **Data Structures (AVL Tree), Sorting Algorithms, and GUI development** in a real-world academic system.

---

## 🚀 Features

- 👨‍🎓 Add student records (Reg No, Name, Semester, Department)
- 📚 Enroll students in courses
- 🌳 AVL Tree for efficient student data storage
- 🔍 Sorted student listing (A–Z) using Selection Sort
- 📊 Course popularity tracking using Merge Sort
- 🖥️ Interactive GUI using Tkinter
- ⚡ Fast searching and structured data handling

---

## 🛠️ Tech Stack

- Python 🐍
- Tkinter (GUI)
- AVL Tree (Self-balancing BST)
- Sorting Algorithms (Selection Sort, Merge Sort)
